This repository is the backend structure for an AI personal finance assistant. It is responsible for managing user financial data, processing transactions, running logic and powering AI diven logic.
This system is built using DJANGO and DJANGO REST FrameworK. 
TECH STACK
Backend Framework : Django,  Django REST Framework.
Database : PostgreSQL.
AI/Data Processing : Python.
Authentication : JSON Web Tokens (JWT).
Dev Tools : GIT
